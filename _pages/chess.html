--- layout: archive permalink: /chess/ title: "My Chess hobbies" author_profile: true --- Chess is a world goverend by a few rules. Enormous complexity rises from this small set of rules which has mesmerised chess theoreticians over centuries. It's been quite a while since the computers surpassed humans in chess. However, the purity of the game has kept it alive as an interesting problem for machine learning researchers. As an amateur chess player and a machine learner, I write here about the reciprocal benefits that chess can bring to machine learning and vice versa.
